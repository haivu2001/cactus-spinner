cactus-spinner
---
A library of loading spinners made with Svelte.

## Example 1
```html
<main>
    <h1>Hello {name}!</h1>
    <p>Visit the <a href="https://svelte.dev/tutorial">Svelte tutorial</a> to learn how to build Svelte apps.</p>
    <RectSpin speed={15} color="white">
        <span>Loading</span>
    </RectSpin>
</main>

```
